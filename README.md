# Codex Prueba

Repositorio de prueba para aprender el flujo de trabajo con Git y GitHub.

## ¿Qué es este proyecto?

Este es mi primer repositorio. Aquí practico cómo crear ramas, hacer commits y abrir Pull Requests.

## Cómo empezar

1. Clona el repositorio
2. Haz tus cambios en una rama nueva
3. Abre un Pull Request

## Contribuir

¿Quieres mejorar este proyecto? Sigue estos pasos:

1. Haz un fork del repositorio
2. Crea una rama: `git checkout -b mi-mejora`
3. Guarda tus cambios: `git commit -m "Descripción del cambio"`
4. Súbelos: `git push origin mi-mejora`
5. Abre un Pull Request desde GitHub
