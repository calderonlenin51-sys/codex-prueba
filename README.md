# Codex Prueba

Repositorio de prueba para aprender el flujo de trabajo con Git y GitHub.

## ¿Qué es este proyecto?

Este es mi primer repositorio. Aquí practico cómo crear ramas, hacer commits y abrir Pull Requests.

## Cómo empezar

1. Clona el repositorio
2. Haz tus cambios en una rama nueva
3. Abre un Pull Request
